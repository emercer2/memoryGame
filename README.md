# Run
cd memoryGame
Add .env file and fill VITE_APP_NOOKIPEDIA_API_KEY with a Valid Nookipedia API Key (https://api.nookipedia.com/)
npm install
npm run dev

OR

Visit https://memory-flax.vercel.app/ where I have it hosted through Vercel.
